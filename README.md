<h2>Description</h2>
This lab demonstrates a basic new hire onboarding process using Active Directory. 
The project focuses on creating a new user account, assigning a temporary password, enforcing a password change at next logon, and verifying proper placement within the appropriate Organizational Unit (OU).
<br />

<h2>Environments Used </h2>

- <b>Windows Server 2022</b>
- <b>Vmware Workstation</b>
